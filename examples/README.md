## What each example demonstrates

- **#1 (human subtitles)** — the ideal case. Clean transcript → richest analysis. This is the
  quality bar to aim for. Prefer videos with human-authored captions.
- **#2 (Shorts / portrait)** — proves the layout adapts to 9:16 vertical video without squashing,
  and that even a 30-second clip yields useful idiomatic content.
- **#3 (noisy auto-captions)** — the tool detects ASR garbling (e.g. `j'ai demandé à Vanessa`
  mis-heard as `j'ai mon bac et vannes`), **drops the unusable segments**, and only mines
  expressions from trustworthy lines — never teaching you broken French.
- **#4 (multi-speaker street interview)** — rich in discourse markers (`voilà`, `hein`, `ben/bah`,
  `perso`) and idiomatic collocations (`se sentir à sa place`, `ça prend la tête`, `il y a rien à
  voir`) that surface naturally in unscripted conversation across several different speakers.

## Regenerate any of these

```bash
export SUPADATA_API_KEY=sd_xxx
python3 scripts/fetch_transcript.py dkU-56i1ZCM --lang fr --out ./work
python3 scripts/merge_sentences.py ./work/dkU-56i1ZCM.transcript.json --out ./work/sentences.json
# → agent mines expressions + translations →
python3 scripts/snap_timestamps.py ./work/expr.json ./work/corpus.json --out ./work/expr.snapped.json
python3 scripts/build_html.py --template assets/template.html --video-id dkU-56i1ZCM \
  --title "…" --header "…" --expressions ./work/expr.snapped.json \
  --vocab ./work/vocab.json --corpus ./work/corpus.json --aspect 16/9 --out french-lab-dkU-56i1ZCM.html
```
